# Namefit N-Constellation Logo Drafts

## Files
- `namefit-n-constellation-v1.svg`: soft and friendly primary draft (5-color nodes)
- `namefit-n-constellation-v2.svg`: structured geometric draft (3-color system)
- `namefit-n-constellation-v3-mono.svg`: monochrome utility draft for small sizes
- `namefit-n-constellation-v4-4dots.svg`: four-dot N draft (no top node)

## Recommended direction
Use `namefit-n-constellation-v4-4dots.svg` as primary brand symbol and keep `namefit-n-constellation-v3-mono.svg` for favicon or stamp contexts.
